## Assemble API

A Rails API for the Assemble mobile app.

To run:

* clone the repo
* run `bundle install`
* run `rails db:migrate`
* run `rails db:setup`
* run `rails server`
* visit `localhost:3000`

Check out the `routes.rb` file and controllers to see the available endpoints.
